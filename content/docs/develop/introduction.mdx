## What is Saazy Template?

Saazy Template is a repository that contains a Next.js app and reusable components for building a beautiful marketing website for SaaS or anything else.

## Highlights

- 12+ beautifully designed pages.
- Fully responsive.
- [Contentlayer](https://www.contentlayer.dev/) based CMS â€“ write **blog** and **docs** with type-safe Markdown/MDX.
- Powered by [Camome UI](https://camome.net).
- Dark mode supported.
- Basic SEO.
- Free updates.

## Tech stacks

- [TypeScript](https://www.typescriptlang.org/) for most code.
- [Next.js](https://nextjs.org/) for framework, **OR** static HTML/CSS/JavaScript.
- [Camome UI](https://camome.net) for UI framework.
- [CSS Modules](https://github.com/css-modules/css-modules) for styling.
- [Contentlayer](https://www.contentlayer.dev/) for processing markdown/mdx files for blog and docs.
- [heroicons](https://heroicons.com/) for most of the icons.
- [pnpm](https://pnpm.io) for package manager.
- [ESLint](https://eslint.org/) for linter.
- [Prettier](https://prettier.io/) for formatter.
