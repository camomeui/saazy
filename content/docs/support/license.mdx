## Overview

The license grants permission to an individual or entity (the Licensee) to access and use the Product.

You **can**:

- Use the Products to create unlimited End Products.
- Use the Products to create unlimited End Products for unlimited Clients.
- Use the Products to create End Products where the End Product is sold to End Users.
- Use the Products to create End Products that are open source and freely available to End Users.

You **cannot**:

- Use the Products to create End Products that are designed to allow an End User to build their own End Products using the Products or derivatives of the Products.
- Re-distribute the Products or derivatives of the Products separately from an End Product, neither in code nor as design assets.

## Terminology

**Licensee** is the individual or entity who has purchased a license.

**Products** are the source code and design assets made available to the Licensee after purchasing a license.

**End Product** is any artifact produced that incorporates the Products or derivatives of the Products.

**End User** is a user of an End Product.

**Client** is an individual or entity receiving custom professional services directly from the Licensee, produced specifically for that individual or entity.
